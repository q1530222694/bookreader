# 不同文件 / 模块 的依赖关系（当前完整版 · 2026-07-14 更新）

> 本文件只保留「最新完整的依赖说明」，每次修改/新增文件后都会就地刷新对应条目。
> 架构铁律见 `docs/提示词.md`：禁止 `packages/` 反向依赖 `lib/`；UI 不硬编码颜色/字号/文案（走 `CupertinoColors.xxx` / `Theme` / `LocalizationEngine.text(...)`）；颜色变更必须上浮全局主题。

## 一、架构总览
- `lib/`（主工程 App）
  - `app/`：启动环境与全局注册中心
  - `core/`：主工程基础设施适配（主题引擎、字体管理）
  - `engine/`：鉴权引擎、服务端配置同步、本地化引擎
  - `features/shell/`：书架 / 阅读 / 回忆 / 设置等业务模块
  - `shared/`：全局 UI 组件
- `packages/`：跨应用复用中台 SDK（会员 / 支付 / 广告），本 App 当前未引入独立 packages 依赖，所有业务均在 `lib/` 内。

### `lib/features/shell/model/reading_stats_model.dart`（阅读统计模型）
- **功能**：由书籍列表聚合阅读统计数据，供回忆页与统计详情页复用。
- `fromBooks(books)`：聚合 `dailyMinutes`（按天）、`dailyBlockMinutes`（`Map<DateTime,List<int>>`，每天 4 个 6 小时段）、`hourlyMinutes`（按时段 0~23）、`dailyHourlyMinutes`（`Map<DateTime,List<int>>`，每天 24 小时档，供「日」视图趋势图/热力图）、**【2026-07-14 三次修改】** `dailyQuarterMinutes`（`Map<DateTime,List<int>>`，每天 96 段 = 24 小时×4 个 15 分钟段，按下标 `小时×4 + (分钟~/15)` 归桶，供「日」视图热力图每格=15 分钟的小方格着色）。
- `completedBooksInRange(books, startInclusive, endExclusive)`：区间内「读完」（`progress>=1.0` 且 `lastReadAt` 落在区间）的书籍数，统一口径。
- `activeDaysInRange(startInclusive, endExclusive)`：**【2026-07-14 新增】** 统计区间内「有阅读活动的天数」（累计阅读天数），供详情页新增指标卡使用。
- `minutesBetween` / `MonthTimelineRecord.monthTimeline(books)`：既有能力，未变。
- 依赖 ➔ `lib/features/shell/model/book_model.dart`
- 被消费 ➔ `memory_main_page.dart` / `memory_page.dart`
