# 不同文件 / 模块 的依赖关系（当前完整版 · 2026-07-13 更新）

> 本文件只保留「最新完整的依赖说明」，每次修改/新增文件后都会就地刷新对应条目。
> 架构铁律见 `docs/提示词.md`：禁止 `packages/` 反向依赖 `lib/`；UI 不硬编码颜色/字号/文案（走 `CupertinoColors.xxx` / `Theme` / `LocalizationEngine.text(...)`）；颜色变更必须上浮全局主题。

## 一、架构总览
- `lib/`（主工程 App）
  - `app/`：启动环境与全局注册中心
  - `core/`：主工程基础设施适配（主题引擎、字体管理）
  - `engine/`：鉴权引擎、服务端配置同步、本地化引擎
  - `features/shell/`：书架 / 阅读 / 回忆 / 设置等业务模块
  - `shared/`：全局 UI 组件
- `packages/`：跨应用复用中台 SDK（会员 / 支付 / 广告），本 App 当前未引入独立 packages 依赖，所有业务均在 `lib/` 内。

## 二、文件依赖明细（当前状态）

### `lib/features/shell/ui/bookshelf_page.dart`（书架主页面）
- **功能**：书架页 `CupertinoPageScaffold`，含顶部统计卡片、搜索、分类标签（全部/PDF/EPUB/TXT/其他）、最近阅读列表、导入与更多操作弹层。
- `_buildStatsCards`：顶部 4 项统计（全部 / 收藏 / 在读 / 已读），由 `_buildStatCell` 渲染；每项数据经构造函数传入、点击无回调（纯展示）。
  - **【2026-07-13 本次修改】**：① 卡片外边框 `Border.all(systemGrey5)` 移除，背景由 `systemBackground` 改为 `secondarySystemBackground.resolveFrom(context)`，阴影调淡为 `primaryColor.withOpacity(0.06)`，整体线框更柔和、与回忆页阅读统计卡片格式一致；② 删除 3 条 `systemGrey5` 灰色竖线分隔 `Container(width:1,height:46)`；③ 首项标签由「全部书籍」改为「全部」（走 `bookshelf_all_label`，其 zh 值已改为 '全部'）。
- `_buildRecentReading`：最近阅读书籍网格，点击跳转对应阅读器（按格式）。
- 依赖/调用 ➔ `lib/features/shell/controller/bookshelf_controller.dart`（书籍数据源 `books`）
- 依赖/调用 ➔ `lib/engine/localization_engine.dart`（全部文案）
- 依赖/调用 ➔ `lib/features/shell/model/book_model.dart`（书籍模型）
- 跳转 ➔ `book_viewer_page.dart` / `epub_viewer_page.dart` / `txt_viewer_page.dart` / `comic_viewer_page.dart`（按格式打开）

### `lib/features/shell/controller/bookshelf_controller.dart`（书架控制器）
- **功能**：书架「聪明大脑」，持有 `books`（`ValueListenable<List<BookModel>>`），负责导入、去重判断、阅读状态更新、进度回写；重复导入用短时 toast 提示。
- 依赖/调用 ➔ `lib/features/shell/service/bookshelf_service.dart`
- 消费 ➔ `lib/features/shell/model/book_model.dart`
- 被消费 ➔ `bookshelf_page.dart` / `memory_main_page.dart` / `memory_page.dart` / `reading_records_page.dart` / `forgotten_books_page.dart` / `book_viewer_page.dart`

### `lib/features/shell/service/bookshelf_service.dart`（书架持久化服务）
- **功能**：维护书架书籍列表与阅读进度（本地持久化）。
- 依赖 ➔ `lib/features/shell/model/book_model.dart`

### `lib/features/shell/model/book_model.dart`（书籍数据模型）
- **功能**：书籍实体（标题、封面、格式、进度 `progress`、是否收藏 `isFavorite`、`lastReadAt`、阅读时长等）。
- 变更 ➔ 增加 `copyWith()`，便于更新进度而不丢失元数据。

### `lib/features/shell/model/reading_stats_model.dart`（阅读统计模型）
- **功能**：由书籍列表聚合阅读统计数据，供回忆页与统计详情页复用。
- `fromBooks(books)`：聚合 `dailyMinutes`（按天）、`dailyBlockMinutes`（`Map<DateTime,List<int>>`，每天 4 个 6 小时段 `[0]=00~06 [1]=06~12 [2]=12~18 [3]=18~24`）、`hourlyMinutes`（按时段）。
- `completedBooksInRange(books, startInclusive, endExclusive)`：区间内「读完」（`progress>=1.0` 且 `lastReadAt` 落在区间）的书籍数，统一口径。
- `MonthTimelineRecord` + `MonthTimelineRecord.monthTimeline(books)`：按 `lastReadAt` 月份聚合（书名数 / 分钟 / 收藏 / 读完 / 开始列表）。
- 依赖 ➔ `lib/features/shell/model/book_model.dart`
- 被消费 ➔ `memory_main_page.dart` / `memory_page.dart`

### `lib/engine/localization_engine.dart`（本地化引擎 · 全局唯一文案来源）
- **功能**：集中式多语言，对外暴露 `LocalizationEngine.text('key')`；含 zh/en 双语。
- **【2026-07-13 本次修改】**：`bookshelf_all_label` 值由 `{'zh':'全部书籍','en':'All Books'}` 改为 `{'zh':'全部','en':'All'}`（书架统计首项文案缩短）。
- 主要相关键（节选）：`bookshelf` / `bookshelf_all_label`(全部) / `bookshelf_tab_all`(全部) / `bookshelf_tab_other` / `bookshelf_favorites_label`(收藏) / `bookshelf_reading_label`(在读) / `bookshelf_finished_label`(已读) / `done` / `view_all`；回忆页与统计页全套键（`reading_statistics` / `stats_tab_week|month|year|all` / `last_year_today` / `random_memory` / `weekly_reading_duration` / `reading_timeline` / `heatmap_*` / `forgotten_*` / `all_reading_records` / `timeline_*` / `weekly_compare_prefix` 等）。
- 被全局所有 UI 调用。

### `lib/features/shell/ui/memory_main_page.dart`（阅读回忆主页面）
- **功能**：长滚动回忆主页，卡片依次为：去年的今天、随机回忆、本周阅读时长（自真实数据+「比上周」）、阅读统计（周/月/年/全部 Tab + 4 项指标 + 右上「统计页」入口）、阅读时间轴（真实月度数据）、阅读热力图（4 象限=4 个 6 小时段、月份选择、主题主色派生色）、遗忘的书籍。
- 背景全部 `CupertinoColors.secondarySystemBackground.resolveFrom(context)`，跟随暗色模式；`_sectionIcon` 统一卡片标题前置图标。
- 依赖/调用 ➔ `bookshelf_controller.dart` / `localization_engine.dart` / `reading_stats_model.dart` / `forgotten_books_page.dart` / `memory_page.dart` / `reading_timeline_page.dart` / `timeline_entry.dart`

### `lib/features/shell/ui/memory_page.dart`（阅读统计详情页 · 二级）
- **功能**：完全按设计稿的「阅读统计详情」页：导航栏、周期选择器（周/月/年/全部，均分胶囊）、区间切换器（周/月/年 上下箭头，「全部」隐藏）、4 项指标卡（时长/本数/页数/笔记）、趋势图（条形/折线切换、纵坐标自适应）、热力图（周/月=4 象限日历、年/全部=GitHub 贡献图）、类型分布甜甜圈、时间分布（时段/频率）、阅读记录（按周期筛选）。
- 颜色全走主题系统、文案全走 `LocalizationEngine`，无硬编码字面量。
- 依赖/调用 ➔ `bookshelf_controller.dart` / `book_model.dart` / `reading_stats_model.dart` / `localization_engine.dart` / `reading_records_page.dart` / 各阅读器页

### `lib/features/shell/ui/forgotten_books_page.dart`（遗忘的书籍二级页）
- **功能**：展示全部未读完书籍（`progress < 1.0`），按未打开天数倒序；响应式 `GridView`；复用 `_openBook` 跳转阅读器。
- 依赖/调用 ➔ `bookshelf_controller.dart` / `book_model.dart` / `localization_engine.dart`

### `lib/features/shell/ui/reading_records_page.dart`（全部阅读记录二级页）
- **功能**：展示所有有阅读记录的书籍（按 `lastReadAt` 倒序）；纯展示子组件 `ReadingRecordRow`（数据经构造传入、点击 `onTap` 回调）。
- 依赖/调用 ➔ `bookshelf_controller.dart` / `book_model.dart` / `localization_engine.dart`

### `lib/features/shell/ui/reading_timeline_page.dart` + `lib/features/shell/ui/timeline_entry.dart`
- **功能**：阅读时间轴「查看全部」页；`TimelineEntry` 无状态条目组件（颜色由调用方以主题派生色传入），`timelineEntryFromRecord` 拼装标题/摘要/详情。
